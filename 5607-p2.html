CSCI 5607 Project 2:
Image Manipulation


In this project, I implemented additional image manipulation routines. Build the executable with the provided Makefile. To reproduce the result images included below, run the included "run" program. Here's an example:

$ make all
$ ./run

There were many aspects of this project that I thought were not very straightforward. I had to spend a lot of time trying different things before settling on a good way to convolve a filter onto the image, and I found it quite hard to wrap my head around the Floyd-Steinberg dither, and was only able to come up with a somewhat flawed implementation. I'm confident I got the sampling routines right; though I'm not able to visually see any distinction between the nearest neighbor and bilinear sampling methods. Pixel operations (most of which were implemented in an earlier assignment) were pretty simple. The compiler's address sanitizer would complain about some of my deletes so I decided to do what any reasonable programmer would do and I left slight memory leaks in some of the functions.

Three sampling methods compared. Left to right, point, bilinear, Gaussian. All images scaled down by 0.5x0.5. The only difference I could notice was that Gaussian made the image slightly blurry while the other two methods kept it reasonably similar.
