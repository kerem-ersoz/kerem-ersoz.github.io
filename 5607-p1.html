CSCI 5607: Project 1

In this project, I implemented some simple image manipulation routines. Build the executable with the provided Makefile (on OSX, use the "mac" target for best results), then run the program "square". Clicking inside the square and dragging will move it, clicking and dragging on an edge will scale it, and clicking and dragging a corner will rotate it. The cursor will change when performing these different transformations. Pressing the "r" key will reset the square's position, size and angle to its defaults. Pressing the spacebar will randomize the scale and angle of the square. The program can optionally be run with commandline arguments to specify additional options. The first argument can be used to specify a path to a .ppm file to load. The second and third arguments enable the increased brightness and triangle rendering features respectively. These arguments are just placeholders, so they do not expect any specific strings. Here's an example:

$ make mac
$ ./square goldy.ppm a a

The most challenging aspect of this project was figuring out the PGA operations needed to robustly implement rotation. I tried several different approaches and ended up using a more conventional arctangent approach to figure out the angles between displacement vectors.
